# HSK 1 · 汉字 Flashcards

A mobile-friendly flashcard and quiz app for studying HSK Level 1 Chinese characters.

## Features
- 160 HSK 1 characters across 16 sets of 10
- Flashcard mode with flip animation
- Quiz mode with A/B/C/D multiple choice (pinyin + meaning)
- Random mix — choose which sets to randomize from
- Score tracking per session

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import your GitHub repo
4. Vercel auto-detects Vite — just click **Deploy**

That's it. No environment variables needed.

## Add to phone home screen

Once deployed, open the Vercel URL on your phone:
- **iPhone**: Share → Add to Home Screen
- **Android**: ⋮ menu → Add to Home Screen
